# NYSTHI-Dark-Theme
A dark theme for NYSTHI's VCV Rack modules. 

# Installation
Rename your res folder in Plugins/NYSTHI/ to res.bak, unzip res-dark.zip and rename to res, move to Plugins/NYSTHI/
